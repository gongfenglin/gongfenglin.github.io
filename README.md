# Gong Fenglin Academic Homepage

This site is based on the MIT-licensed
[luost26/academic-homepage](https://github.com/luost26/academic-homepage)
template, using the layout shown at
[luost.me/academic-homepage/index_layout2](https://luost.me/academic-homepage/index_layout2).

## Local Preview

```bash
bundle install
bundle exec jekyll serve
```

## Content

- Profile data: `_data/profile.yml`
- Navigation: `_data/navigation.yml`
- News: `_news/`
- Publications: `_publications/`
- Blog posts: `_posts/`
- Public files: `files/`

Put downloadable files under:

- `files/cv/`
- `files/papers/`
- `files/slides/`

## GitHub Pages

The intended site URL is:

```text
https://gongfenglin.github.io
```

For that exact URL, the GitHub username or organization must be `gongfenglin`,
and the repository must be named `gongfenglin.github.io`.
